# 03 - Grain Jar Lid Assembly

Grain spawn requires high passive gas exchange to colonize properly. The primary focus of a grain lid is airflow without allowing airborne contaminants inside.

## Layout Options
* **Agar-to-Grain (Filter Only):** If you only use agar wedges to inoculate grain, you only need filter patches. Drill 2 holes and apply 2 patches. (If using high-flow Microppose filters, 1 is sufficient).
* **Syringe-to-Grain (Filter + Port):** If you plan to inject liquid culture directly into the grain, drill 1 hole for a filter patch and 1 hole for a self-healing injection port.

## Drilling Procedure
1.  Screw a lid onto an empty jar for stability.
2.  Drill **1/4-inch holes** for the filter patches.
3.  If adding an injection port, drill a **7/16-inch hole**.
4.  **Important:** Ensure the center of any hole is at least **3/4 of an inch away from the very edge** of the lid so it does not interfere with the sealing ring.

## Assembly
1.  Ensure there are no plastic/metal burrs. Clean up the hole with an Exacto blade if need be.
2.  Wash all lids to remove debris and let them dry completely.
3.  If using an injection port, push it in first. It will be a snug fit; pulling it through from the other side helps. Ensure it is fully seated and level.
4.  Place the adhesive filter patches over the 1/4-inch holes.
