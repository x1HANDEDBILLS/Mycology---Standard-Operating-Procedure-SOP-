# 03 - Jar Lid Assembly

## Layout Options
* **Filter Only:** If using cheaper filter patches, drill 2 air exchange holes and use 2 patches. If using name-brand Microppose filters, 1 filter is all that is needed.
* **Filter + Injection Port:** Use 2 filters and an injection port in a triangle pattern. (Injection ports are only needed if planning to use liquid culture syringes).

## Drilling Procedure
1.  Screw a lid onto a jar for stability.
2.  Drill **1/4-inch holes** for the filter patches.
3.  Drill **7/16-inch holes** for the injection ports.
4.  **Important:** Try to keep the holes evenly spaced apart and make sure the center of the hole is at least **3/4 of an inch away from the very edge** of the lid.

## Assembly
1.  Ensure there are no plastic burrs. Clean up the hole with an Exacto blade if need be.
2.  Wash all lids to remove plastic pieces and let them dry completely.
3.  Push injection ports in first. It will be a snug fit; pulling it through once it has started helps. Make sure it is fully sealed and level.
4.  Place the adhesive filter patches over the remaining holes.
